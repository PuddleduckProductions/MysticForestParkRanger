Class InteractionBehavior

A serialized class meant to control different interaction behaviors when space is pressed on one. This is to avoid having to attach multiple monobehaviors for anything with one interaction. If you want to add your own, you can either use Custom (in progress), or create a subclass of InteractionBehavior.
